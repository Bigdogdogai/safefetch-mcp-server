# Contributing

Thanks for your interest in contributing.

## Before You Start

- Open an issue first for major changes.
- Keep pull requests focused and testable.
- Follow existing code style and update docs when behavior changes.

## Contribution Rules

By submitting code, documentation, tests, or other contributions to this repository, you agree that:

1. You have the right to submit the contribution.
2. Your contribution can be distributed under the project's open-source license (**AGPL v3.0**).
3. The maintainers may also include your contribution in commercial editions under the project's dual-licensing model.

If you do not agree to these terms, please do not submit contributions.

## Recommended PR Checklist

- [ ] Added or updated tests
- [ ] Updated docs/README if needed
- [ ] No secrets, tokens, or local absolute paths included
- [ ] Confirmed JSON output contract remains compatible
